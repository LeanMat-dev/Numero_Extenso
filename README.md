# 🔢 Numero por Extenso (Python)

Projeto em Python que converte números inteiros em sua forma por extenso em português.

---

## Funcionalidades

- Converte números de 0 a 9999
- Trata unidades, dezenas, centenas e milhares
- Suporte a números especiais (11 a 19)
- Lógica baseada em listas e manipulação de strings

---

## O que foi praticado

- Listas em Python
- Indexação de caracteres
- Lógica de decomposição de números

---

## Como executar

```bash
python numero_extenso.py
